üì¶CSS Box Model

Every HTML element is essentially a rectangular box with four main components:

Interactive Box Model Visualization
CONTENT
Margin - Outer space
Border - Element edge
Padding - Inner space
Content - Actual content

üìè Box Model Properties

.element { /* Content dimensions */ width: 200px; height: 100px; /* Padding - space inside the element */ padding: 20px; /* All sides */ padding: 10px 20px; /* Vertical | Horizontal */ padding: 10px 15px 20px 25px; /* Top | Right | Bottom | Left */ /* Border - edge of the element */ border: 5px solid #3498db; border-width: 2px; border-style: solid; border-color: #2980b9; /* Margin - space outside the element */ margin: 20px; /* All sides */ margin: 10px auto; /* Vertical | Horizontal (auto centers) */ margin: 5px 10px 15px 20px; /* Top | Right | Bottom | Left */ }

üìê Box Sizing

box-sizing: content-box vs border-box
/* Default - width/height applies to content only */ .content-box { box-sizing: content-box; /* Default */ width: 200px; /* + padding + border = total width */ } /* Modern approach - width/height includes padding and border */ .border-box { box-sizing: border-box; width: 200px; /* Total width including padding and border */ }
üéØ Pro Tip: Use box-sizing: border-box for more predictable layouts. Many developers set this globally with * { box-sizing: border-box; }

üßÆ Calculation Formula:
Total Width = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right
Total Height = height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom

Common Display Values

display: none; /* Element not displayed */ display: block; /* Block-level element */ display: inline; /* Inline element */ display: inline-block; /* Inline but with block properties */ display: flex; /* Flexbox container */ display: grid; /* Grid container */ display: table; /* Table-like behavior */

Box Model Shortcuts

/* Margin shortcuts */ margin: 10px; /* All sides */ margin: 10px 20px; /* Vertical | Horizontal */ margin: 10px 20px 30px; /* Top | Horizontal | Bottom */ margin: 10px 20px 30px 40px; /* Top | Right | Bottom | Left */ /* Padding shortcuts (same pattern) */ padding: 15px 25px; /* Border shortcuts */ border: 2px solid #333; /* Width | Style | Color */ border-radius: 10px; /* Rounded corners */
üé® Design Tips:
  • Use margin: 0 auto; to center block elements
  • Negative margins can overlap elements
  • inline-block elements have a small gap - use font-size: 0 on parent to remove
  • Collapsing margins: vertical margins between elements can merge
